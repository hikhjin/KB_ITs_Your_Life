CSS 선택자

- 특정한 HTML 태그를 선택하여 스타일을 부여할 때 활용
- 선택자 종류: 태그, #id, .class, 후손, 자산, 이벤트 등등

1. 모든 선택자

- 모든 태그에 스타일을 적용할 때 사용(모든 태그 선택)

2. 태그 선택자 **

- 특정 태그에 스타일을 적용할 때 사용

    • 3. id 선택자 **

    - 특정 id 속성을 가진 태그에 스타일을 적용할 때 사용
    - id와 style가 1:1로 맵핑 될 때만 사용 가능
    - id는 일반적으로 javascript에서 사용되는 식별자로 가급적 css에서는 사용을 꺼림

    - 여러 태그에 동시에 적용 가능, 1:N / 1:1 매핑 가능
